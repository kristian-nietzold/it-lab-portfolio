

## Overview
This lab simulates a real small business environment using Google Workspace and demonstrates practical administration skills relevant to IT support and junior system administrator roles.

---

## Lab Structure

This lab covers the following areas:

- Domain Setup & DNS (MX, SPF, DKIM)
- Organizational Units (OUs)
- User Management (creation, suspension, lifecycle)
- Groups (communication and access control)
- Admin Roles & Permissions
- Security (2-Step Verification, password policies)
- Gmail Configuration (spam filtering, routing basics)
- Google Drive & Sharing (permissions, shared drives)
- Device & Endpoint Management (basic policies)
- Reporting & Monitoring (audit logs, login activity)
- Calendar & Meet configuration
- Basic Compliance (DLP awareness and Vault basics)

---

## Sections

### Domain Setup
Domain verification and DNS configuration.

### Organizational Units (OUs)

Delegating admin access securely.

### Gmail Configuration
Spam filtering and basic routing.

### Google Drive & Sharing
Access control and shared drives.

### Device Management
Basic endpoint control and policies.

### Reporting & Monitoring
Audit logs and activity tracking.

### Compliance (Basic)
DLP awareness and Vault basics.

## Goal of This Lab

To demonstrate:
